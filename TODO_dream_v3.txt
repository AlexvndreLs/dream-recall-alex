[ ] 0b. Vérifier équilibre HR/LR des rejets EOG
        Objectif : s'assurer que le rejet EOG n'est pas systématiquement plus
        fort chez HR que chez LR (ou inverse), ce qui biaiserait l'interprétation
        biologique. Si p > 0.05 → OK, mentionner quand même dans le papier.

        cd ~/dream-recall-alex && source /home/alouis/mne_env/bin/activate
        python3 -c "
        import csv, numpy as np
        from collections import defaultdict
        from config_v3 import HR_SUBJECTS, LR_SUBJECTS
        rows = list(csv.DictReader(open(
            '/home/alouis/scratch/dream_bids/derivatives/ica/ica_rejection_scores.csv')))
        by_sub = defaultdict(list)
        for r in rows: by_sub[r['subject']].append(r)
        hr = [sum(int(r['eog_rejected']) for r in by_sub[f'{s:02d}'])
              for s in sorted(HR_SUBJECTS) if f'{s:02d}' in by_sub]
        lr = [sum(int(r['eog_rejected']) for r in by_sub[f'{s:02d}'])
              for s in sorted(LR_SUBJECTS) if f'{s:02d}' in by_sub]
        print(f'HR (n={len(hr)}): mean={np.mean(hr):.2f} std={np.std(hr):.2f}')
        print(f'LR (n={len(lr)}): mean={np.mean(lr):.2f} std={np.std(lr):.2f}')
        from scipy.stats import mannwhitneyu
        stat, p = mannwhitneyu(hr, lr)
        print(f'Mann-Whitney U p={p:.3f}  (>0.05 = pas de biais groupe)')
        "

[ ] 1b. Vérifier forme et SPD-itude de CoSpectra

        Pourquoi : TSclassifier Riemannien exige des matrices SPD (symétriques
        définies positives). Si CoSpectra retourne des matrices non-SPD,
        le classifieur plante silencieusement ou avec une erreur cryptique.

        Test rapide (nœud de calcul) :
            python3 -c "
            from pyriemann.estimation import CoSpectra
            import numpy as np
            from numpy.linalg import eigvalsh
            X = np.random.randn(10, 19, 7500)  # 10 epochs, 19 canaux, 30s à 250Hz
            csp = CoSpectra().fit_transform(X)
            print('shape:', csp.shape)
            # attend (10, 19, 19, n_freqs) ou (10, n_freqs, 19, 19)
            # vérifier SPD sur première epoch, première fréquence :
            mat = csp[0, 0] if csp.ndim == 4 and csp.shape[1] == 19 else csp[0, :, :, 0]
            print('SPD ?', (eigvalsh(mat) > 0).all())
            "
