    EOG — corrélation absolue, seuil 0.6 (tranché par 3 méthodes convergentes) :
    - Méthode 1 : balayage empirique CV inter-sujets (analyze_thresholds.py)
      Zone plate CV ~0.35-0.45 de 0.2 à 0.5, explosion au-delà de 0.6.
    - Méthode 2 : coude géométrique (intersection de tangentes sur courbe CV)
      → 0.598. Même logique que courbe de titration en chimie.
    - Méthode 3 : coefficient de Gini robuste (corrige le biais CV = σ/μ
      mécaniquement petit quand μ grand) → 0.573.
    - Résultat : 3 méthodes convergent sur 0.57-0.60 → seuil fixé à 0.6
      (valeur ronde, défendable dans le papier, non significativement différent
      de 0.573 ou 0.598 sur ces données).
    - Données finales @ 0.6 : 1.21 rej/sujet, CV=0.66, 15.8% sujets à 0 rejet.

        Après batch final, vérifier que le nombre de rejets EOG ne diffère pas
        significativement entre groupes HR (s1-18) et LR (s19,20,23-38).
        Un déséquilibre = biais potentiel dans l'interprétation biologique.

            python3 -c "
            import csv, numpy as np
            from collections import defaultdict
            from config_v3 import HR_SUBJECTS, LR_SUBJECTS
            rows = list(csv.DictReader(open(
                '/home/alouis/scratch/dream_bids/derivatives/ica/ica_rejection_scores.csv')))
            by_sub = defaultdict(list)
            for r in rows: by_sub[r['subject']].append(r)
            hr = [sum(int(r['eog_rejected']) for r in by_sub[f'{s:02d}']) for s in sorted(HR_SUBJECTS) if f'{s:02d}' in by_sub]
            lr = [sum(int(r['eog_rejected']) for r in by_sub[f'{s:02d}']) for s in sorted(LR_SUBJECTS) if f'{s:02d}' in by_sub]
            print(f'HR: mean={np.mean(hr):.2f} std={np.std(hr):.2f}')
            print(f'LR: mean={np.mean(lr):.2f} std={np.std(lr):.2f}')
            from scipy.stats import mannwhitneyu
            stat, p = mannwhitneyu(hr, lr)
            print(f'Mann-Whitney p={p:.3f}')
            "
